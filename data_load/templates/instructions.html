How to use this application

English

  1. Load operations -> Load tags:

    Loads tags and categories of the books, but not the books

    This is non destructive and takes a couple of seconds

  2. Load operations -> Load reviewers:

    It will make a full load of the reviewers information and a partial load of reviews and books

    Options to load all or only the unchecked

    This will take about 5min if you are loading all

  3. Load operations -> Load reviews:

    Loads full information of the books and reviews

    This can take upto 5h as it loads the book information and the review information

    Tags can only be assigned if the have been loaded first

    The scrape order is based on when it was last updated from oldest to newest

    Books and reviews can be individually updated by from their review details page

    You can take a look on the maximum number of scrapable objects in the "List operations -> Stats" page

  4. Load operations -> Load whoosh:

    Loads the whoosh index with the information in the database

Español

  1. Load operations -> Load tags:

    Carga las etiquetas o categorías de los libros, pero no los libros

    Esto no es destructivo y toma unos segundos

  2. Load operations -> Load reviewers:

    Esto tomará alrededor de 5 minutos si estás cargando todos

    Realizará una carga completa de la información de los revisores y una carga parcial de revisiones y libros

  3. Load operations -> Load reviews:

    Carga la informacón completa de las revisiones y los libros

    Esto puede tardar hasta 5 horas, ya que carga la información del libro y de la revisión

    Las etiquetas solo pueden asignarse si se han cargado primero

    El orden del scraping se basa en cuándo se actualizó por última vez, de lo más antiguo a lo más nuevo

    Los libros y revisiones se pueden actualizar individualmente desde su página de detalles de revisión

    Puedes ver el número máximo de objetos scrapeables en la página "List operations -> Stats"

  4. Load operations -> Load whoosh:

    Carga el indice the whoosh con la información guardada en la base de datos
